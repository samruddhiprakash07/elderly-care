# Elderly AI Assistant

Project scaffold for an AI assistant to support elderly users with reminders, fall detection, and caregiver coordination.

## Structure

- backend: AI logic, APIs, models
- frontend: web UI + dashboard
- docs: pitch, architecture, project overview

## Quickstart

1. Start backend
   - `cd backend`
   - `python -m venv venv`
   - `venv\Scripts\activate`
   - `pip install -r requirements.txt`
   - `python app.py`
2. (Optional) Run voice assistant
   - `python ..\voice_assistant.py`
3. Start frontend
   - `cd frontend`
   - `npm install`
   - `npm start`
